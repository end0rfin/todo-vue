## Скрипты

В проекте доступны базовые скрипты запуска и сборки:

### `yarn dev`

Запуск проекта в development-режиме - [http://localhost:3000](http://localhost:3000).

### `yarn test`

Запуск unit-тестов проекта.

### `yarn build`

Сборка проекта в production-режиме.
